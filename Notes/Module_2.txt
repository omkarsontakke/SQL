What is ALTER Command?
Ans:- Alter command is used to do the modification/changes in the existing table.

What is the Difference between DELETE, TRUNCATE and DROP
Ans:-
    DELETE command is used to delete the schema and data. *We can ROLLBACK*
    TRUNCATE command is used to delete the data inside the table, not the table
    DROP command is used to DELETE COMPLETE TABLE

What is the best approach to find null value in column?
Ans:- Use the IS Filtering pattern instead of '='.
    SELECT * FROM Emp_Table WHERE email='null'; Wrong
    SELECT * FROM Emp_Table WHERE email IS null; Right

What is the difference between CHAR And VARCHAR
